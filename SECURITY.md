# Security Policy

## Supported Versions

This project is currently developed on a best-effort basis.

## Reporting a Vulnerability

Please do not open public issues for security reports.

- Prefer GitHub Security Advisories ("Report a vulnerability").
- If that is not available, open a private communication channel with the maintainers.

## Sensitive Data

- Never include API keys, access tokens, or `.coplan/auth.json` contents in bug reports.
- OpenAI key usage should default to `OPENAI_API_KEY` environment variable.

For additional notes, see `docs/security.md`.
